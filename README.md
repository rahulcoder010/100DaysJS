# 100DaysJS
100 Days to JS Master

Welcome to the 100 Days to JS Master challenge!

In this challenge, you will spend 100 consecutive days learning and mastering JavaScript. Each day, you will complete a small coding exercise or project to enhance your skills and understanding of JavaScript.

To participate in the challenge, follow these instructions:

1. Fork this repository.
2. Clone the forked repository to your local machine.
3. Create a new branch for each day's exercise or project.
4. Complete the exercise or project for the day.
5. Commit and push the changes to your branch.
6. Submit a pull request to the main repository.

By the end of the 100 days, you will have a solid foundation in JavaScript and be ready to take on more advanced projects.

Get started now and become a JavaScript master in just 100 days!

Good luck!
